Bài 1
git init
Khởi tạo kho Git trong thư mục hiện tại (tạo thư mục ẩn .git/). Từ đây có thể add/commit/push.

git add <file_name>
Đưa một file cụ thể từ Working Directory vào Staging Area (Index) để chuẩn bị commit.

git add .
Đưa tất cả thay đổi trong thư mục hiện tại và con của nó (thêm/sửa; không stage file đã xóa nếu phiên bản Git cũ).
Mẹo an toàn hơn: git add -A sẽ stage cả thêm/sửa/xóa.

git commit -m "message"
Tạo một ảnh chụp (commit) từ những gì đang ở Staging Area, kèm thông điệp mô tả thay đổi.
